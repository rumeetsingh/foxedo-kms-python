# foxedo_kms_python
Access your FoxedoKMS keys using this package

## Installation
```python
pip install FoxedoKMS
```

## Import foxedo_kms
```python
from foxedo_kms import keys
```

## Usage
```python
fox = keys.Foxedo('Your Access Key')
my_key = fox.get_key('Key Id')
```
